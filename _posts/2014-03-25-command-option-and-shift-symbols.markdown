Unicode macOS symbols list:

* ⌘ – `&#x2318;` – `&#8984;` – Command Key
* ⌥ – `&#x2325;` – `&#8997;` – Option Key
* ⇧ – `&#x21E7;` – `&#8679;` – Shift Key
* ⎋ – `&#x238B;` – `&#9099;` – ESC Key
* ⇪ – `&#x21ea;` – `&#8682;` – Capslock
* ⏎ – `&#x23ce;` – `&#9166;` – Return
* ⌫ – `&#x232b;` – `&#9003;` – Delete

To be able use this kind of notes: `⌘ + ⇧ + ⏎`
